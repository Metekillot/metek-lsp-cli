# metek-lsp-cli

## It ain't much, but it's performant

A convenience tool for interacting with the LSP protocol from the command line ergonomically and performantly.

It's a hack and a half of scripts to generate the methods, and some butchery of Omnisharp/csharp-language-server-protocol in order for it to get running on .NET 10.0 and .NET 11.0, but it's both faster and easier to use than all the Python and JS/TS implementations of the same kind of tool I was able to find.

### ./lib
Egregious 'cp -r' of Omnisharp/csharp-language-server-protocol, then whatever modifications were necessary for it to compile into .NET 10/11. 

### ./Driver
Ergonomic handle on under-the-hood Juju of Omni#/c#lsp, which is itself wrapped by LspWrapper

### ./LspWrapper
Wrapper on the wrapper; a PowerShell handle and interface to streamline things even further. Driver is prefectly servicable as a tool for the purpose if you're not inclined to `pwsh`.

### ./generated
Intended to be a drag+drop smorgasbord of LSP methods for you to add/remove from Driver as pleases you. Slapping them all on at once makes it horrible to use in the CLI, so I tucked them in here instead. Many of them were generated by a script, but work without issue with some tweaking, depending on your use case.

### ./pwsh_handles/generate_methods.ps1
The script I vibeslopped then rewrote then vibe sanded to generate the methods after I got sick of reflection-based method discovery. This is where the sausage got made.
